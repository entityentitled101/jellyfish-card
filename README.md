# 趋光性 Photo taxi

一只笨拙的机器水母产品卡

**Live site:** https://cathayandy.github.io/jellyfish-card/

---

## 关于

这是一个为 Hackathon 2025 项目「趋光性」设计的四页折叠产品卡。视觉风格极其简洁优雅：

- 纯黑背景 + 白色细线框
- 衬线字体（中文：Noto Serif SC，英文：Cormorant Garamond）
- SVG 线条插图装饰
- 正方形尺寸，精致诗意

## 四页内容

1. **封面** - 趋光性 Photo taxi
2. **内页左** - 起源故事
3. **内页右** - 照料指南
4. **封底** - 它在找光，也许也在找你

## 技术栈

- HTML5 + CSS3
- 原生 JavaScript
- 响应式设计
- 零依赖

## 本地预览

```bash
open index.html
```

或使用本地服务器：

```bash
python3 -m http.server 8000
```
